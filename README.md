# Penetration-Testing-Laboratory-VAPT-Reports
Repository containing penetration testing coursework, VAPT reports, PoC exploits, and supporting material developed during the analysis of intentionally vulnerable systems and applications. Covers the full assessment workflow: reconnaissance, vulnerability assessment, exploitation, post-exploitation, and remediation.

## Contents

- Final penetration testing reports
- Vulnerability assessment findings
- Exploitation and post-exploitation evidence
- PoC payloads and test scripts
- Network scanning and enumeration results
- Supporting screenshots and documentation

## Methodology

The activities documented in this repository follow a standard penetration testing workflow:

1. Information Gathering  
2. Network Scanning  
3. Enumeration  
4. Vulnerability Assessment  
5. Exploitation  
6. Post-Exploitation  
7. Reporting & Remediation

## Disclaimer

All activities were performed in controlled laboratory environments or intentionally vulnerable systems for educational purposes only. No unauthorized testing was conducted against real-world targets.
